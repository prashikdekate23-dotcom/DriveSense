# 🚗 DriveSense

## Smart Automotive Dashboard

A modern automotive digital dashboard prototype developed using HTML, CSS and JavaScript.

Designed to simulate real-time vehicle parameters such as:

- Vehicle Speed
- Engine RPM
- Fuel Level
- Engine Temperature
- Battery Voltage
- Odometer
- Vehicle Status
- System Warnings
- Live Digital Clock
